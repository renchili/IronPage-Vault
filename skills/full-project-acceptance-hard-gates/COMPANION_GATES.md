# Companion Gates Index

This directory contains the `full-project-acceptance-hard-gates` skill and mandatory companion gates.

When using `SKILL.md`, also apply every companion gate in this directory:

```text
documentation-pollution-gates.md
real-interaction-testing-gates.md
```

These files are not optional examples. They are part of the acceptance methodology.

A final PASS is not allowed if a target project contains relevant documentation or interaction surfaces and the corresponding companion gate tables are omitted.
