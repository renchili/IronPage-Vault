Recheck P0 fix scope

This branch fixes the static recheck blockers without writing directly to main:

- run_tests.sh prepares generated Swagger before compiling or testing.
- ci/run_tests_contract_check.sh verifies the local entrypoint from a clean docs/swagger state.
- ci/swagger_contract_check.sh verifies every route-level @Router annotation appears in generated swagger.yaml and checks key response contracts.
- Pull Request CI runs local-entrypoint and swagger contract jobs when their owning files change.
- docs/requirement-check.md separates PR CI, merge candidate regression, post-merge evidence, and local/manual replay.
- docs/swagger-artifacts.md documents generated Swagger artifact strategy.
